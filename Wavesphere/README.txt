This is the Wavesphere embedded application master project.

Some rules for committing:
1. Do not commit the debug folder. This is important. Don't do it.
2. Never commit a project that does not build. Never. Ever.
3. The project does not have to be fully working for it to be commited. But please MAKE SURE IT
	BUILDS.

Some rules for editing:
1. Use tabs for indenting. DO NOT USE SPACES.
2. Do not allow a line to go over 120 columns in length.
3. Remove all trailing whitespace.
